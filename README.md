# Game Introduction

這是一款物理益智遊戲，玩家需在螢幕繪圖（如線條、重物），利用重力與槓桿原理達成特定目標（如推倒物體），挑戰以最少筆劃過關，若超時則會失敗。

[遊戲連結](https://www.youtube.com/watch?v=DLSxpePtxss)
